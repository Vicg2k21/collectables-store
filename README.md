# 1/6 Scale Collectables Store Application

## 📌 Overview
This is a desktop-based online store application developed using C# and Visual Studio 2022. The system supports multiple user roles including managers, employees, and customers, allowing for full store functionality such as browsing products, managing inventory, processing orders, and generating reports.

## 🚀 Features

### 🔐 User Authentication
- Login as Manager, Employee, or Customer
- Account registration
- Password recovery / reset functionality
- Option to continue as a guest

### 🛒 Shopping Experience
- Browse 1/6 scale collectible items with images
- Add items to cart
- Apply promo codes for discounts
- Checkout process with order receipt generation

### 📦 Order Management
- View order history (customers & employees)
- Track completed purchases

### 🧑‍💼 Manager Features
- Manage user accounts
- Manage inventory and products
- Create and manage discounts
- Generate business reports
- Perform POS (Point of Sale) checkout for customers

### ❓ Help System
- Built-in help button available on all forms/screens for user assistance

## 🛠 Technologies Used
- C#
- .NET Framework
- Windows Forms (GUI)
- Visual Studio 2022
- Object-Oriented Programming (OOP)
- SQLite (DB Browser for SQLite)

## ▶️ How to Run
1. Open the solution file (`.sln`) in Visual Studio 2022
2. Build the project
3. Run the application
4. Login, register, or continue as a guest to explore features

## 📂 Project Structure
- `.sln` → Solution file
- Project folders → Source code and forms
- `Resources/` → Images and assets
- `CollectablesDB.db` → SQLite database file used by the application
